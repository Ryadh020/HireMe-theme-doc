I. Installation

Step 1: WordPress installation

To install this theme, you must have a working version of WordPress already installed on your server. WordPress is well-known for its ease of installation.
https://codex.wordpress.org/Installing.WordPress. Please visit this link for installing WordPress.

Step 2: Installation Through Dashboard

  1. Log in to the WordPress Dashboard.
  2. Go to Appearance->Themes.
  3. Click on Add New Button.
  4. Search HireMe
  5. Click on Install tab and Activate it
  6. Now your HireMe theme is ready to Use

II. Bottom Header Setup

Setup Menu

  1. Go To Apperance > Menu
  2. Create a new menu by clicking on create a new menu
  3. Add your menu items as pages/ Category/ Custom Url in left side menu
  4. Choose “Main Menu” from below Checkbox

III. Social widget area setup

Step1: Setup Social links

  1. Go To Apperance > Menu
  2. Create a new menu by clicking on create a new menu

  3. Add your menu items as Custom links in left side menu

Step2: Put links on the widget area

  1. Next, Go To Widgets > select "navigation menu" > select "social media widget area" > click "add widget"

IV. Setup Frontpage

Step1: Home page

  1. Go to Dashboard > Pages > Add new

  2. Create Home Page
  3. choos a thumbnail

Step2: Blog page

  1. Go to Dashboard > Pages > Add new
  2. Create blog Page

Step3: Setup reading

  1. Go to Settings > Reading and select Front page displays > A static page (select “Home”) & Posts page ( “Blog”)

V. Setup work page

Step1: Create the page with "work template"

  1. Go to Dashboard > Pages > Add new

  2. Create Home Page
  3. Go to Settings > page attributes > template > select work template

Step2: Add work Categories to the page

  1. Next, Go To appearance > go to Widgets > select "Categories" > select "categories widget area" > click "add widget"
